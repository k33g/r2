#R2 Documentation

R2 is a very simple class helper to make ajax queries.

##Initialize R2

    import r2 from './r2';

##GET

    new r2.Request("/humans").get().then((data) => {
      console.log(data);
    })

    new r2.Request("/humans/hFPDywuRsPdJM7Fa").get()
      .then((data)=>{
        console.log(data);
      })
      .catch((error) => {
        console.log(error);
      });
